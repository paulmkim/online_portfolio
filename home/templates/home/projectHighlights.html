Deterlab Security Suite


The Deterlab Security Suite consists of multiple different security-focused simulations. It covers many real-life scenarios and challenges the user to use real tools to solve issues.

  • POSIX Permissions and Stateful Firewalls
    • POSIX server hardening using permissions
    • Modifying a stateful firewall for network hardening
  • Exploits
    • Attacked a web server using buffer overflow, pathname traversal, and SQL injection
    • Created patches for found vulnerabilities
  • Computer Forensics
    • Discovering the mode of attack on a server through evidences such as logs, recovered memory, and images
    • Repairing damages and preventing future attacks
    • Counterattack using recovered information about attacker
  • Network Attacks
    • Analyzing a network using a protocol analyzer (Wireshark)
    • Attacking through ARP Spoofing, replay, insertion, and encryption attacks
    • Repairing network protocol to prevent future attacks
  • Botnets and Network Analysis
    • Network analysis for unusual behaviors
    • Detection and prevention of botnet attacks


Mini Java Compiler


Mini Java is a subset of java. It contains all functions of java except overloading. Using a visitor design pattern, the Mini Java Compiler was capable of verifying that a program was valid Mini Java, and compile the code down to assembly language (MIPS).



Prereq


Prereq is a web application built on Django, with a MySQL database and hosted on AWS. It allows users to learn any desired topic though a dynamic knowledge tree with subnodes consisting of topics that are required before you can master the desired topic. For example, if I wished to learn Algebra, I would be presented with subnodes with resources to learn arithmetics, multiplication/division, and fractions.



NOMS


NOMS is a mobile application built for the Android, with a MySQL database and hosted on Aws. It allows users to plan short term events with ease. Those who are invited can quickly assess the event and respond, while the host can have a simple overview of the current status of the event. It differs from most event planning appliction because it focuses on events with short notice, such as dinner, which allows the interface to remove many complexities.



LoLCounter


LoLCounter is a web application built on Ruby on Rails, with a MySQL database and hosted on AWS. It creates a place for League of Legends players to see champion statistics based on millions of real games. This project focused on maintaining functionality and availability against high traffic and large databases (Although we were limited to using MySQL).



Object Rect


ObjectRect is a fully automated camera man. It uses FLIR thermal imaging camera, which is connected to a Raspberry Pi. The Raspberry Pi used the input from the termal camera to locate a preventors location, and communicated with multiple servos attached to the camera to follow the presentors.



Exploring the Development of TCP/IP


This paper explored the history and development of the layered network system. It looks into the original intent and current status of the layered design. In addition, it specifically looks into why TCP/IP was eventually split into two layers and conclude on the benefits of this split. The research was conducted through reports of multiple first-hand developers.
